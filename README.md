# Full-Stack Mastery: 40-Project Engineering Residency

Welcome to my public learning and portfolio repository! This space tracks my progression through 40 distinct full-stack applications, moving from modular monolithic architecture up to advanced cloud-native distributed systems and AI integration workloads.

## Technical Ecosystem Covered
* **Frontend:** Semantic HTML5, CSS3 (Flexbox/Grid), Modern TypeScript, Angular (17/19), Reactive State (RxJS)
* **Backend:** C# 12/13, ASP.NET Core 8/9, Minimal APIs (Carter), MediatR (CQRS Pattern)
* **Databases:** SQL Server, Dapper Micro-ORM, Entity Framework Core
* **Advanced Systems:** JWT Security, Redis Caching, RabbitMQ Message Brokers, Azure Cloud Services, Semantic Kernel AI

## Storage Efficiency Workflow
To maintain a high-performance workspace under local disk storage constraints, this repository utilizes an **Incremental Scaffolding Workflow**:
1. Build and test the full-stack system locally.
2. Commit and push the complete source code history to GitHub.
3. Execute `rm -rf **/bin **/obj **/node_modules` to instantly reclaim local disk space before initializing the next challenge blueprint.
