# Full-Stack Mastery: 20-Project Placement & Engineering Residency

Welcome to my public portfolio repository! This space tracks my progression through 20 specialized full-stack applications. The first block is explicitly engineered to master the logic routines, Object-Oriented Programming (OOP) milestones, and advanced SQL architectures required to clear high-tier fresher technical screening exams (such as the TCS NQT Advanced Coding Round and IBM Associate System Engineer assessments).

## 🚀 The Core Technical Stack
* **Frontend:** Semantic HTML5, CSS3 (Flexbox/Grid), Modern TypeScript, Angular (17/19), Reactive State (RxJS)
* **Backend:** C# 12/13, ASP.NET Core 8/9, Minimal APIs (Carter), MediatR (CQRS Pattern)
* **Databases & Logic:** SQL Server, Dapper Micro-ORM, Entity Framework Core, Data Structures & Algorithms (DSA)
* **Cloud & AI:** Redis Caching, RabbitMQ Message Brokers, Azure Cloud Integrations, Semantic Kernel (AI Vector RAG Workloads)

## 📦 Storage Efficiency & Local Disk Safeguard
To maintain a high-performance workspace under strict local disk storage limitations (40 GB free space restriction), this repository enforces a compiled artifact disposal strategy:
1. Build, optimize, and execute the complete full-stack layer locally.
2. Commit and push the lightweight text source code configuration history safely to GitHub (tracked via a robust `.gitignore` filter).
3. Run `rm -rf **/bin **/obj **/node_modules` to instantly reclaim physical drive space before initializing the next challenge block.
